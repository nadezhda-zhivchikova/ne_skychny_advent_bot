## Advent Telegram Bot

Python‑бот для адвент-календаря: игры, рецепты и маршруты, рассылаемые раз в день.

### Возможности
- **Ежедневная рассылка** одного сообщения адвента по подписке.
- **Период работы**: автоматически активен с 26 декабря по 11 января включительно.
- **Приветствие** при первом запуске с предложением подписаться.
- **Админ-интерфейс** через команды Телеграма для заполнения и редактирования адвент-календаря.
 - **Хранение данных** в локальных JSON-файлах (`data/users.json`, `data/advent_days.json`).

### Установка
1. Установите Python 3.10+.
2. Перейдите в каталог проекта:
   ```bash
   cd advent_bot
   ```
3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

### Настройка
1. Создайте бота через BotFather и получите токен.
2. Узнайте свой Telegram user ID (например, через бота `@userinfobot`) — он будет админом.
3. Откройте файл `config.py` и задайте значения:
   ```python
   BOT_TOKEN = "ВАШ_ТОКЕН_ЗДЕСЬ"
   ADMIN_USER_ID = 123456789
   DAILY_SEND_HOUR = 10
   TIMEZONE = "Europe/Moscow"
   ```

### Запуск
```bash
cd advent_bot
python main.py
```

Бот должен быть запущен постоянно (на сервере / VPS / хостинге), чтобы рассылка работала каждый день.

### Админ-команды
- **/admin_help**: показать доступные админ-команды.
- **/admin_add_day YYYY-MM-DD | Заголовок | Описание**: создать или обновить запись адвента на указанную дату.
- **/admin_list_days**: список всех заполненных дней адвента.
- **/admin_broadcast_today**: принудительно отправить адвент за сегодня всем подписчикам.
